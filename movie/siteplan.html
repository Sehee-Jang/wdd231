Project Site Plan

Site Name

CineQuest

This name was chosen because the website will allow users to explore movies by searching and filtering based on different criteria. The name 'Quest' reflects the idea of embarking on a journey to discover new films and information about them.

Site Purpose

The website's primary purpose is to allow users to search for movies, view details like cast, release date, and reviews, and filter movies based on genre or rating. The site will also provide movie trailers and recommendations.

Scenarios

1. A user wants to search for a specific movie and see its details.

2. A user wants to browse movies based on their favorite genre.

Color Schema

Secondary Color: #f0f4f8 (Used for background)

Typography
